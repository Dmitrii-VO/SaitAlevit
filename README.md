# Сайт строительной компании "АЛЕВИТ СТРОЙ"

Современный лендинг для строительной компании с адаптивным дизайном, калькулятором стоимости, картой объектов и интеграцией с Telegram-ботом.

## 🚀 Функциональность

*   **Адаптивная верстка:** Корректное отображение на мобильных, планшетах и десктопах (Mobile-First подход).
*   **Калькулятор:** Расчет предварительной стоимости дома на основе площади, материалов и этажности.
*   **Интерактивная карта:** Яндекс.Карты с отображением построенных и строящихся объектов.
*   **Галерея:** Просмотр фото проектов и выполненных работ.
*   **Telegram Бот:** Админ-панель для управления контентом сайта (проекты, работы, отзывы) через Telegram.
*   **Оптимизация:** Все изображения автоматически конвертируются в WebP для ускорения загрузки.

## 🛠️ Технологии

*   **Frontend:** HTML5, CSS3 (BEM, CSS Variables), Vanilla JavaScript (ES6 modules).
*   **Backend:** Node.js (для скриптов сборки и бота).
*   **Tools:** Webpack, PostCSS, Babel, Jest, ESLint, Prettier, Sharp.

## 📦 Установка и Запуск

1.  **Установка зависимостей:**
    ```bash
    npm install
    ```

2.  **Запуск в режиме разработки:**
    ```bash
    npm run dev
    ```
    Сайт будет доступен по адресу `http://localhost:3000`.

3.  **Сборка для продакшена:**
    ```bash
    npm run build
    ```
    Эта команда соберет HTML, CSS и JS файлы, оптимизируя их для выкладки.

4.  **Запуск Telegram бота:**
    ```bash
    npm run bot
    ```
    Не забудьте настроить `.env` файл с токеном бота (см. `telegram-bot/.env.example`).

## ✅ Тестирование и Качество кода

В проекте настроены инструменты для обеспечения качества кода:

*   **Unit-тесты:** Запуск тестов для утилит и компонентов.
    ```bash
    npm test
    ```
*   **Линтинг:** Проверка JavaScript кода на ошибки.
    ```bash
    npm run lint
    ```
*   **Форматирование:** Автоматическое исправление стиля кода.
    ```bash
    npm run format
    ```

## 🖼️ Оптимизация изображений

Все изображения в проекте хранятся в папке `images/` и автоматически конвертируются в формат **WebP** для максимальной производительности.

*   **Скрипт оптимизации:** `npm run optimize-images` (запускает `node scripts/optimize-images.js`).
*   **Скрипт очистки:** `node scripts/cleanup-images.js` (удаляет исходники `.png`/`.jpg`, если есть `.webp`).

## 📁 Структура проекта

*   `src/` - Исходный код (HTML partials, CSS компоненты, JS модули).
*   `data/` - JSON файлы с данными (проекты, работы, отзывы).
*   `images/` - Изображения (автоматически оптимизируются).
*   `scripts/` - Служебные скрипты (сборка, сервер, оптимизация).
*   `tests/` - Тесты Jest.
*   `telegram-bot/` - Логика бота.
